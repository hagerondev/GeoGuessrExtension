# GeoGuessrExtension

これはGeoGuessrをより楽しくプレイするためのGoogle Chrome Extensionです。

## NEWS

- Chromeウェブストアにて審査中

## 機能

- 全員同時スタート
- リアルタイムのランキング更新
- 残り時間によるスコア加算


## アーキテクチャ

- Amazon API Gatewayのserverless websocket
- DynamoDB
- Lambda
